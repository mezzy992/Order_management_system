# Order Management System

Java-based ERP Order Management System built using **JSP, Servlets, JDBC, and MySQL**.

## Features

* Seller Dashboard
* Product Management (Add / Update / Delete)
* Order Management
* Reported Products Monitoring
* Seller Profile Management
* Login & Registration System

## Tech Stack

* Java
* JSP
* Servlets
* JDBC
* MySQL
* Bootstrap
